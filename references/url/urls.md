# 良デザイン参考URLリスト

業種ごとに「これを参考にしたい」というデザインの優れたサイトURLをタグ付きで蓄積する。

Claude Code は HTML 生成前にこのファイルを読み、該当業種のURLが必要なら WebFetch で取得して参考にする。

---

## 記入フォーマット

```markdown
- [サイト名](URL) `#業種タグ` — メモ（何が良いか）
```
