# 介護業界デザインルール

このファイルは Claude Code が介護業のサイトを生成する前に必ず読み込む。
配色・フォント・トーン・必須セクション・禁止事項を厳守する。

---

## 配色（プレースホルダ：実運用前に確定する）

| 役割 | hex コード | 用途 |
|------|-----------|------|
| メインカラー | `#（後で決定）` | 主要な見出し・ボタン・アクセント |
| サブカラー | `#（後で決定）` | セクション背景・補助要素 |
| 背景 | `#FFFFFF` または淡い暖色 | ページ全体の背景 |
| テキスト | `#333333` | 本文 |
| アクセント | `#（後で決定）` | CTA ボタン |

候補：
- 「安心・温かみ」系：ベージュ系 + サーモンピンク
- 「清潔・信頼」系：淡いブルー + ホワイト + 木の色
- 「自然・癒し」系：セージグリーン + クリーム

---

## フォント

| 用途 | フォント | 備考 |
|------|---------|------|
| 見出し | Noto Sans JP（700） | Google Fonts で読み込み |
| 本文 | Noto Sans JP（400） | 行間 1.8、文字間 0.05em |
| 数字（電話番号等） | システム sans-serif | 読みやすさ優先 |

---

## トーン（伝えるべき情緒）

- **安心**：「ここなら任せられる」と感じさせる
- **信頼**：実績・スタッフの姿が見える
- **温かみ**：家族のように寄り添う
- **誠実**：誇張表現を避け、事実を丁寧に伝える

---

## 必須セクション

ヒアリング確定の3セクション。

1. **ヒーロー**
   - キャッチコピー（介護施設らしい温かみのある一文）
   - サブコピー（具体的なサービス／所在地）
   - メインビジュアル（施設・スタッフ・利用者の笑顔）
   - CTA（電話 or 問い合わせ）

2. **サービス内容・実績**
   - 提供サービス一覧（訪問介護・デイサービス・ショートステイ等）
   - 各サービスの簡単な説明
   - 実績数字（運営年数・利用者数など、ある場合）

3. **アクセス・CTA**
   - 住所・最寄り駅
   - 電話番号（大きく表示）
   - 営業時間
   - 問い合わせフォーム or 電話誘導

---

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 派手な装飾（ネオン色・グラデーション過多） | 信頼性を損なう |
| 暗色基調 | 介護業は明るい印象が必須 |
| 機械的な表現（「最先端AIで」「革新的な」等） | AI臭・冷たさが出る |
| 過度な擬人化（「私たちの想い…」を多用） | 安っぽくなる |
| 絵文字 | 軽い印象になる |
| 高齢者を「お年寄り」と呼ぶ | 「ご利用者様」「ご入居者様」を使う |

---

## アニメーション指針

- スクロール時のフェードイン：ゆっくり（0.6〜0.8秒）
- ホバー時の変化：穏やか（色の濃淡変化のみ。回転や拡大は避ける）
- 派手な視差効果（パララックス）は使わない

---

## レイアウト指針

- 余白は十分に取る（セクション間 80〜120px）
- 1カラムベースで縦に流す（横並び多用しない）
- 写真は人物中心。風景・建物は補助
- フォントサイズ：見出し 32〜48px、本文 16〜18px（高齢者向けにやや大きめ）
